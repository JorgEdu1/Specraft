Informaçoes Relevantes para o uso adequado do Site:

O uso do site é orientado por seguintes classificaçoes:


Basico: Para jogos triplo A, rodar com taxa de 60FPS em uma qualidade grafica baixa
Para jogos competitivos, uma taxa de 100FPS em uma qualidade grafica baixa


Moderado: Para jogos triplo A, rodar com taxa de 60FPS em uma qualidade grafica mediana
Para jogos competitivos, uma taxa de 144FPS em uma qualidade grafica mediana


Basico: Para jogos triplo A, rodar com taxa de 60FPS em uma qualidade grafica alta
Para jogos competitivos, uma taxa de 240FPS em uma qualidade grafica alta
